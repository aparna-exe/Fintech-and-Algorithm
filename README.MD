#  Fintech & Algorithm 

A collection of interactive applications focused on financial modeling, debt optimization, and algorithmic data visualization. This suite demonstrates the bridge between complex back-end logic and modern UI/UX.

---

##  Project Directory

| Project | Description | Tech Stack |
| :--- | :--- | :--- |
| **[FutureWealth Tracker](./FutureWealth%20Tracker)** | 30-year compound interest forecasting with real-time graph updates. | `Python`, `Matplotlib`, `CustomTkinter` |
| **[Smart Splitter](./Smart%20Splitter)** | A greedy-algorithm engine to minimize transactions in group debt settlement. | `Python`, `CustomTkinter` |
| **[Sorter](./Sorter)** | An interactive visualizer for data organization and time complexity. | `Python`, `CustomTkinter` |

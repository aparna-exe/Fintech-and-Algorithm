#  Fintech & Algorithm 

A collection of interactive Python applications focused on financial logic and data visualization.This repository is where I explore the intersection of financial logic and algorithmic efficiency with a heavy focus on building clean and modern UI that make complex data easy to digest.

---

##  Project Directory

| Project | Description | Tech Stack |
| :--- | :--- | :--- |
| **[FutureWealth Tracker](./FutureWealth%20Tracker)** | Predicts where your money will be in next coming years using real time math. | `Python`, `Matplotlib`, `CustomTkinter` |
| **[Smart Splitter](./Smart%20Splitter)** | Transaction optimization for group debt settlement. | `Python`, `CustomTkinter` |
| **[Sorter](./Sorter)** | An interactive visualizer for data organization and time complexity. | `Python`, `CustomTkinter` |

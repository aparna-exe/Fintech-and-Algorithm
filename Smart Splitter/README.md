#  Smart Splitter

**Smart Splitter** Smart Splitter is a debt-optimization tool designed to simplify group expenses. Instead of a messy web of everyone owing everyone, this app calculates the most efficient way to settle all debts in the fewest possible transactions.
---

###  How it's calculated
The app calculates settlements by first determining the Total Group Cost and dividing it by the number of participants to find each person's Equal Share. It then compares what each person actually spent against that share
